## 1. 产品概述

家电保养提醒工具——帮助用户管理家中各类电器的清洁与保养计划，通过智能提醒避免遗忘，延长电器使用寿命。

- 目标用户：普通家庭用户，拥有多台家电需定期保养维护
- 核心价值：告别"上次清洗是什么时候"的遗忘困境，用系统化的方式追踪保养周期，到期自动预警

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 录入电器、记录保养、查看提醒 |

### 2.2 功能模块

1. **仪表盘页面**：今日提醒概览、即将到期电器列表、保养日历视图
2. **电器管理页面**：电器列表（按下次保养日期排序）、添加/编辑/删除电器
3. **保养记录页面**：历史保养记录、操作人与花费追踪

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘 | 状态概览 | 显示已到期、7天内到期、安全的电器数量统计 |
| 仪表盘 | 提醒列表 | 展示即将到期和已过期的电器，按紧急程度排序 |
| 仪表盘 | 快捷操作 | 一键标记已保养、快速添加新电器 |
| 电器管理 | 电器列表 | 所有电器卡片，显示名称、类型、下次保养日期、状态标签 |
| 电器管理 | 添加电器表单 | 录入电器信息，支持预设保养模板（空调、洗衣机等） |
| 电器管理 | 编辑/删除 | 修改电器信息和保养周期，删除电器 |
| 保养记录 | 记录列表 | 历史保养记录时间线，包含操作人、花费、备注 |
| 保养记录 | 新增记录 | 记录本次保养的操作人（自己/维修工）、花费金额、备注 |

## 3. 核心流程

用户打开应用 → 仪表盘展示到期/即将到期提醒 → 点击"立即保养"记录保养操作 → 系统自动计算下次保养日期 → 电器状态更新为"安全"

用户添加电器流程：点击"添加电器" → 选择电器类型（可选预设模板）→ 输入电器名称 → 设置上次保养日期和保养周期 → 系统自动计算下次保养日期 → 保存到列表

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖橙色（#F97316）作为主色调，传达温暖与提醒感；深灰色（#18181B）作为背景基调
- **辅助色**：翠绿色（#22C55E）表示安全/正常状态，琥珀色（#F59E0B）表示即将到期，红色（#EF4444）表示已过期
- **按钮风格**：圆角胶囊按钮，带微妙阴影和hover缩放效果
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**：卡片式布局，顶部导航栏，左侧图标导航
- **图标风格**：使用 Lucide 图标库的线性风格图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表盘 | 状态概览 | 三个统计卡片（已过期/即将到期/安全），渐变背景，大数字+图标 |
| 仪表盘 | 提醒列表 | 紧急度色条卡片，左侧色带标识状态，右侧快捷操作按钮 |
| 仪表盘 | 快捷操作 | 浮动圆形按钮，点击展开添加表单 |
| 电器管理 | 电器列表 | 网格卡片布局，卡片含电器图标、名称、倒计时天数、状态徽章 |
| 电器管理 | 添加表单 | 模态框，分步骤引导，预设模板快速选择 |
| 保养记录 | 记录时间线 | 左侧时间轴，右侧卡片，包含操作人头像/图标、花费金额标签 |

### 4.3 响应式设计

- 桌面端优先：三列网格布局，侧边导航
- 平板适配：两列布局，顶部导航
- 移动端适配：单列布局，底部标签导航
